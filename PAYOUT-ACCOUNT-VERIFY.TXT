Validate Account
This endpoint is used to validate an account based on the provided account number and IFSC code. It is particularly useful for transactions that require account verification before proceeding.

Request
Method: POST
URL: https://api.sparkuptech.in/api/dto/validate_account

Request Body
Field	Type	Description
purpose_message	String	Message stating the reason or note for the transaction. Here, "This is a penniless transaction" indicates itâ€™s only for validation, not money transfer.
validation_type	String	Type of validation being performed. "penniless" means only account verification without transferring funds.
account_number	String	The bank account number to be validated. Masked here for security.
ifscCode	String	The IFSC (Indian Financial System Code) of the bank branch where the account is held. Masked here for security.
HEADERS


partnerid:240054
consumerkey:b2078d92ff9f8e9e
consumersecret:ba6fba9775548f71

curl --location 'https://api.sparkuptech.in/api/dto/validate_account' \
--header 'partnerid:  240054' \
--header 'consumerkey: b2078d92ff9f8e9e' \
--header 'consumersecret: ba6fba9775548f71' \

    "account_number": "1011461xxxx",

}'
